# 📦 Packaging

## 🩵 Go 🩵 project structure generator

### 🛠️ Installation

🔵 1. Clone the repo:
`shell
    git clone https://github.com/admodev/go-packaging && cd go-packaging
    `

🔵 2. Execute the following command:
`shell
    make install
    `

🔵 3. If you need help, run:
`shell
    packaging --help
    `

🔵 4. Enjoy your new project structure ;)
